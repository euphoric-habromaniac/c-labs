## how to run  
  
prerequisites:  
  
* a c compiler (e.g., gcc)  
  
steps:  
   
1. compile a program  
   `gcc exercises/1.average-of-two-numbers.c -o average`  
2. run the program  
   `./average`  
  
repeat for other programs by replacing the filename.
