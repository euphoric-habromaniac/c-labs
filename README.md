## How to Run

**Prerequisites:**  
- A C compiler (e.g., gcc)

**Steps:**  

1. Compile a program:  
   ```bash
   gcc lab_1/print_welcome.c -o welcome
   ```

2. Run the program:  
   ```bash
   ./welcome
   ```

Repeat the process for other programs by replacing the filename.
